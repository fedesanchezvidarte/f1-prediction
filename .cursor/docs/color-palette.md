# Color Palette

This document describes the color palette of the application.

## Palette

- #CF2637 - Crimson Red
- #2A2B2A - Graphite Black
- #FFFFFF - White
- #A06CD5 - Lavender Purple
- #FFB100 - Amber Flame
- #44AF69 - Jungle Green
- #3C91E6 - Bright Ocean Blue

## Usage

- #CF2637 - Crimson Red will be used for the main brand color.
- #2A2B2A - Graphite Black will be used for the background color.
- #FFFFFF - White will be used for the text color.
- #A06CD5 - Lavender Purple will be used for the accent color.
- #FFB100 - Amber Flame will be used for the error color.
- #44AF69 - Jungle Green will be used for the success color.
- #3C91E6 - Bright Ocean Blue will be used for the info color.

Note: we didn't develop the color palette yet, but we will start designing the UI as a dark theme, so the colors will be adjusted later. But we will keep this palette as a starting point.
