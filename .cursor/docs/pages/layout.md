# Layout

## Top Navigation

- Only for logged in users.
- Avatar of the user in the top right corner and in the left of the avatar will be the name of the user.
  - The avatar will be a circular image of the user.
  - The avatar will be display a drawer menu with the following options:
    - Profile: will redirect to the profile page.
    - Language selection (English / Spanish). --> We will implement this later.
    - Theme selection (Dark / Light). --> We will implement this later.
    - Sign out in the bottom of the menu.
- Link to the home page (app logo) in the top left corner and at the right of the link will be the app name and the season year --> Example: F1 Prediction (bold) Season 2026 (regular, small text, gray color and same line), this last part will be only visible in web.
- Mobile view will be only [App logo - User name - Avatar that opens the drawer/hamburger menu].
- Desktop view: will be [App logo - app name and season year - User name - Avatar that opens the drawer/hamburger menu].

## Footer

- Only for logged in users.
- Footer will be at the bottom of the page and will have the following content:
  - Divider line with the same color as the background.
  - App logo and app name and season year.
  - Copyright text with the year of the current year.
  - Link to my LinkedIn profile.
  - Link to my GitHub profile.
  - Link to the privacy policy page. --> We will implement this later.
  - Link to the terms of service page. --> We will implement this later.
  - Link to the contact us page. --> We will implement this later.
- Mobile view will be only [Copyright text with the year of the current year - Link to my LinkedIn profile - Link to my GitHub profile].
- Desktop view will be [Copyright text with the year of the current year - Link to my LinkedIn profile - Link to my GitHub profile - Link to the privacy policy page - Link to the terms of service page - Link to the contact us page].
- The footer will be fixed at the bottom of the page and will not scroll with the content.
